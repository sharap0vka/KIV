concepts / рабочий глоссарий

38 entries · cc by 4.0
01

vibe-coding.

Разработка через диалог с моделью: код — побочный продукт хорошо сформулированного намерения, редактор — собеседник, не транскрайбер.

aliases · pair-prompting · intent-first
 02

agentic loop.

Цикл «думай → выбери tool → выполни → наблюдай», обёрнутый в бюджет токенов и персистентный трейс. Базовая единица любого автономного агента.

see also · react-loop · scratchpad
 03

MCP.

Model Context Protocol — JSON-RPC поверх stdio/SSE, который описывает tools, resources и prompts; превращает «контекст» в подключаемый интерфейс.

aliases · model-context-protocol
 04

cursor-skill.

Декларативная единица знания агента: триггер, входы, инструкции. Скилл — это функция, промпт — её stack-trace при вызове.

see also · prompt-template · macro
 05

RAG.

Retrieval-Augmented Generation. Поиск релевантного куска перед генерацией. Чаще всего — самый дорогой способ заменить хороший индекс файлов.

aliases · grounding · context-injection
 06

tool-calling.

Способ модели запросить структурированное действие наружу: имя функции + JSON-аргументы. Граница между «текстом» и «программой».

aliases · function-calling · structured-output

сайт, который сам себя документирует.

КИВ — это персональный digital-garden и одновременно его исходный код. Источник истины — git: статьи и концепции в MDX, тренды в JSON, дизайн-система в одном файле. Ни БД, ни CMS, ни runtime-AI. Cursor-агенты пишут материалы через PR, GitHub Action ежедневно собирает снимок трендов и коммитит его в репо.
